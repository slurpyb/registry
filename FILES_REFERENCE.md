## Build Pipeline Commands

### Full Rebuild
```bash
cd /Users/jordan/j/src/ocx-slurpyb

# 1. Extract metadata
bun scripts/build-skill-index.ts

# 2. Classify skills (requires CHUTES_API_TOKEN)
CHUTES_API_TOKEN=... bun scripts/classify-skills.ts

# 3. Generate registry
bun scripts/generate-registry.ts

# 4. Generate bundles
bun scripts/generate-bundles.ts

# 5. Build
bun run build

# 6. Deploy
bun run deploy
```

### Individual Steps
```bash
# Just rebuild index
bun scripts/build-skill-index.ts

# Just classify (resume from last checkpoint)
CHUTES_API_TOKEN=... bun scripts/classify-skills.ts --resume

# Just generate registry
bun scripts/generate-registry.ts

# Just generate bundles
bun scripts/generate-bundles.ts

# Just build
bun run build

# Just deploy
bun run deploy
```

### Dry Runs
```bash
# Preview registry without writing
bun scripts/generate-registry.ts --dry-run

# Preview classification prompt
bun scripts/classify-skills.ts --dry-run

# Preview bundles without writing
bun scripts/generate-bundles.ts --dry-run
```
